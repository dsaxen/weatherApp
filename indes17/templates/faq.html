Frequently asked questions

Q: No weather data is shown for my city. Why?

A: This site uses openweathermap.org as the weather data provider. The service does not support every city in the world, only the relatively big ones.

Q: Why should I register to this site?

A: By registering, you can store your favorite locations and language preference. The application will be much faster to use when you don't need to type anything in order to check the weather.

Q: Is this site secure?

A: Yes. This site uses SSL as security protocol. Pythonanywhere.com provides SSL automatically for sites with their domain name.

Q: What are the limitations of the API?

A: The API can handle at most 60 requests per minute, so let's hope this site does not get popular.

Q: I added three favorite cities, but I cannot add more. Why?

A: You are limited to three favorite cities, since the API can only handle 60 requests per minute.
